# Команды Git, использованные в проекте

git init
    Создание локального репозитория.

git add .
    Добавление всех файлов в индекс.

git commit -m "Добавлен SRS и UML"
    Фиксация изменений.

git branch feature/uml
    Создание новой ветки для UML.

git checkout feature/uml
    Переключение на ветку.

git merge feature/uml
    Слияние ветки с основной.

git remote add origin <URL>
    Привязка локального репозитория к GitHub.

git push -u origin main
    Загрузка файлов на GitHub.
